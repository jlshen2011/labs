# streamlit-home

## Requirements
The following packages are required:
* altair
* streamlit
* mortgage

Make sure you have them before running the code. 

## Installation

```bash
git clone https://github.com/jlshen2011/streamlit-home.git
cd streamlit-home
streamlit run app.py
```
